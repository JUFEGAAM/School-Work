Extras

.length: Cuenta cuántos elementos hay dentro del array

.join(","): Toma todos los elementos del array y los une en un solo texto, separados por una coma

.push(): Agrega un nuevo elemento al final del array

.unshift(): Agrega un nuevo elemento al principio del array

.includes(): Revisa cada elemento del array

.shift(): Elimina el primer elemento del array

.pop(): Elimina el ultimo elemento del array

+=: Agrega texto nuevo al final de lo que ya tenías guardado

/n: Salto de linea

Math.max(...): Lee los numeros del array con los 3 puntos, sin ellos no funciona
