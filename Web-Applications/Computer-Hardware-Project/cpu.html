CPU (CENTRAL PROCESSING UNIT)

Model: Intel Core i5-2320

Definition & Function

The CPU is the core of the PC. It works like a brain, executing instructions and bringing order to all the data. Without it, the computer wouldn't have the workflow needed to function correctly.

Assembly & Disassembly

Step 1: PREPARATION

First, to reach the socket, unplug the computer and open the case. Next, remove the CPU Cooler. In this specific model, instead of screws, simply turn the locking pins at the four corners to release the mechanism. Once the cooler is off, the CPU is revealed.

Step 2: REMOVING THE COMPONENT

Now, locate the retention lever that locks the CPU in place. Push it down, move it slightly to the side to unhook it, and let it rise slowly. This releases the metal cover. Finally, carefully lift the CPU out of the socket, holding it only by the edges.

Step 3: INSTALLING THE NEW COMPONENT

For the final step, we have to be careful. First, check if you have a PGA (pins on CPU) or LGA (pins on socket) type. This is important so you don't bend the pins. To install, look for the corner with a small triangle mark on both the CPU and socket. That mark tells you the correct orientation. Place the CPU slowly and very carefully. Once placed, put on the metal cover and push down the lever. If it fits perfectly, you did it right; but if you placed it wrong, that is when you will hear a very painful noise.

Maintenance & Tips

For CPU maintenance, the main task is changing the thermal paste. It is recommended to change it every 2 to 3 years, or if you notice higher temperatures than usual. Also, remember to clean the dust from the heatsink and fan regularly to ensure good airflow
